# PythonArduinoGUI

## Описание проекта

Arduino LED Dimmer — это графическое приложение на Python для управления яркостью светодиода (LED) через плату Arduino. Приложение предоставляет интуитивный интерфейс с ползунком для плавной регулировки яркости, кнопками для мгновенного включения/выключения, а также поддержкой горячих клавиш.

## Основные возможности

- Плавная регулировка яркости светодиода с помощью ползунка.
- Кнопки для мгновенного включения (100%) и выключения (0%) светодиода.
- Поддержка горячих клавиш:  
  - `1` — включить светодиод.  
  - `0` — выключить светодиод.  
  - `Ctrl+Q` — закрыть приложение.
- Настройка задержки (в секундах) через поле ввода.
- Информационное окно с версией программы и описанием.
- Защищённое закрытие приложения с подтверждением.

## Требования

- Python 3.x
- Установленные библиотеки:
  - `tkinter` (обычно входит в стандартную поставку Python)
  - `pyfirmata`
- Плата Arduino (Uno, Nano или аналогичная), подключённая по COM-порту.
- Установленная на Arduino стандартная прошивка Firmata (например, через `StandardFirmata` из примеров Arduino IDE).

## Настройка и запуск

1. Подключите Arduino к компьютеру и определите номер COM-порта (например, `COM6`).
2. Загрузите в Arduino скетч `StandardFirmata` через Arduino IDE.
3. В файле `mat.py` измените строку подключения `Arduino("COM6")` на ваш COM-порт.
4. Установите зависимости:  
   ```bash
   pip install pyfirmata
   ```
5. Запустите приложение:  
   ```bash
   python mat.py
   ```

## Структура интерфейса

Приложение состоит из трёх основных блоков:

- **Настройки** — поле для ввода времени задержки.
- **Управление Яркостью** — горизонтальный ползунок для регулировки и две кнопки "ВКЛ"/"ВЫКЛ".
- **Системные кнопки** — кнопки "Info" (о программе) и "Exit" (выход).

## Примечания

- Если Arduino не подключён, программа выведет сообщение об ошибке, но интерфейс будет запущен для тестирования (закомментирован выход по ошибке).
- Для корректной работы убедитесь, что на пинах 3 и 5 Arduino подключён светодиод через резистор.
- Приложение не поддерживает изменение размеров окна.
